The launch of ChatGPt certainly changed all our lives, and it has been the talk of the tech world ever since!

This project dives into Twitter/X data to train Machine Learning models using regression and classification for identifying the number of likes and likelihood of receiving high likes for a given tweet.

Regression: OLS Backward Stepwise Regression

Classification Models:

  • Decision Tree
  • Logistic Regression
  • K Nearest Neighbors
  • Support Vector Machine
  • Naive Bayes
  • Random Forest
  • AdaBoost
  • Neural Network - MultiLayer Perceptron
