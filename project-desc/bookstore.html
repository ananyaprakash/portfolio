Here's an end-to-end website I built using a robust technology stack including React and TypeScript for the client-side and Java for the server-side, with MySQL for database management, all deployed on AWS.

The website has multiple pages from home page with new releases, category pages for each category, and cart page and a checkout and transaction page. The website was designed on Figma and the SQL-based backend stores books, categories and cart information.

The website uses advanced web development functionalities such as RESTful APIs, React hooks, form validation, and context storage to enhance user experience and efficiency.
